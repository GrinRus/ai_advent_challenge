# Implementation Plan — TST-001: FS-RBAC-03 Live RBAC Enforcement

**Status:** READY
**Ticket:** TST-001
**Updated:** 2026-03-18
**Research Status:** warn (see research findings for PRD overrides)

---

## Executive Summary

This plan implements live RBAC enforcement for all mutation APIs (POST/PUT/DELETE) with audit logging and frontend role synchronization. Based on research findings, we adapt the PRD assumptions to the actual codebase:

| PRD Assumption | Research Finding | Adaptation |
|----------------|------------------|------------|
| JWT claims auth | X-Profile-Key header auth | Custom WebFlux filter with header parsing |
| Redux state | Zustand with persistence | Extend Zustand store with SSE listener |
| WebSocket push | SSE already exists for flows | Extend SSE for role notifications |
| Spring Security | Not in classpath | Custom WebFlux filters (no Spring Security) |

---

## AIDD:DECISIONS

### Architecture Decisions

1. **No Spring Security** — Adding spring-boot-starter-security introduces significant complexity and potential conflicts with existing WebFlux setup. Instead, implement custom `WebFilter` for RBAC.

2. **Extend SSE instead of WebSocket** — SSE infrastructure already exists (`subscribeToFlowEvents` in apiClient.ts). Adding WebSocket requires new dependencies and infrastructure. SSE is sufficient for unidirectional server-to-client role updates.

3. **Role Caching with Caffeine** — Backend uses Caffeine cache (already in classpath). Cache profile roles with 5-minute TTL to meet P95 ≤ 10ms latency requirement.

4. **Audit Log as Separate Table** — Follow existing pattern (note_entry table) with PostgreSQL JSONB for flexible metadata.

5. **Header-Based Identity** — Use `X-Profile-Key` (format: `namespace:reference`) and `X-Profile-Auth` headers for identity extraction.

---

## AIDD:NON_NEGOTIABLES

- All POST/PUT/DELETE APIs must return 403 for unauthorized requests
- Audit log must capture: timestamp, actor profile key, action (GRANT/REVOKE/DENIED), target resource, result
- Frontend role guards must update within 5 seconds of backend role change
- RBAC check latency P95 ≤ 10ms
- Zero breaking changes to existing API contracts

---

## AIDD:RISKS

| Risk ID | Description | Probability | Impact | Mitigation |
|---------|-------------|-------------|--------|------------|
| R-001 | Profile service unavailable | Medium | High | Circuit breaker; fail-closed; cache roles |
| R-002 | Cache stale roles after assignment | Medium | Medium | Cache invalidation on role change; TTL 5min |
| R-003 | SSE connection memory leak | Low | Medium | Proper cleanup on disconnect; subscriber limits |
| R-004 | Audit log write failures | Low | Medium | Async logging; error queue; never block request |
| R-005 | RBAC filter performance degradation | Low | High | Caffeine cache; async audit; P95 target 10ms |
| R-006 | Breaking change to existing APIs | Low | High | Feature flags; audit-only mode first; rollback |

### Validation-Derived Risks (from TST-001-validation.md)

The application uses profile-based component scanning. The RBAC module needs to be added to ALL profiles or a shared configuration created.
- **Mitigation**: Create `RbacConfig.java` with `@Profile("!test")` active in all non-test profiles

**R-008: Profile Service Integration**
The plan assumes `profileClient.fetchRoles()` exists, but no backend API for fetching profile roles was found in research.
- **Mitigation**: Define profile service contract or implement local role storage via `rbac_profile_roles` table

#### 1.3 RBAC Filter (WebFlux)
**File:** `backend-mcp/src/main/java/com/aiadvent/mcp/backend/rbac/filter/RbacWebFilter.java`

```java
@Component
@Order(Ordered.HIGHEST_PRECEDENCE + 100) // After logging, before business logic
public class RbacWebFilter implements WebFilter {

    private final ProfileRoleService roleService;
    private final AuditLogService auditService;
    private final RbacPathMatcher pathMatcher;

    @Override
    public Mono<Void> filter(ServerWebExchange exchange, WebFilterChain chain) {
        HttpMethod method = exchange.getRequest().getMethod();
        String path = exchange.getRequest().getPath().value();

        // Skip non-mutation methods
        if (!isMutationMethod(method)) {
            return chain.filter(exchange);
        }

        // Skip excluded paths (health, public APIs)
        if (pathMatcher.isPublicPath(path)) {
            return chain.filter(exchange);
        }

        // Extract identity from headers
        ProfileIdentity identity = extractIdentity(exchange.getRequest());
        if (identity == null) {
            return auditAndDeny(exchange, null, path, method, "Missing identity headers");
        }

        // Check required roles for path
        Set<Role> requiredRoles = pathMatcher.getRequiredRoles(path, method);

        return roleService.hasAnyRole(identity, requiredRoles)
            .flatMap(hasRole -> {
                if (hasRole) {
                    auditService.logAccess(identity, path, method, "ALLOWED");
                    return chain.filter(exchange);
                } else {
                    return auditAndDeny(exchange, identity, path, method, "Insufficient permissions");
                }
            })
            .onErrorResume(e -> {
                log.error("RBAC check failed", e);
                return auditAndDeny(exchange, identity, path, method, "RBAC error: " + e.getMessage());
            });
    }

    private Mono<Void> auditAndDeny(ServerWebExchange exchange, ProfileIdentity identity,
                                     String path, HttpMethod method, String reason) {
        auditService.logDenied(identity, path, method, reason);
        exchange.getResponse().setStatusCode(HttpStatus.FORBIDDEN);
        exchange.getResponse().getHeaders().setContentType(MediaType.APPLICATION_JSON);
        String body = String.format("{\"error\":\"Forbidden\",\"reason\":\"%s\"}", reason);
        DataBuffer buffer = exchange.getResponse().bufferFactory().wrap(body.getBytes());
        return exchange.getResponse().writeWith(Mono.just(buffer));
    }
}
```

#### 1.4 Profile Role Service with Caching
**File:** `backend-mcp/src/main/java/com/aiadvent/mcp/backend/rbac/service/ProfileRoleService.java`

```java
@Service
public class ProfileRoleService {

    private final LoadingCache<ProfileIdentity, Set<Role>> roleCache;
    private final ProfileClient profileClient;

    public ProfileRoleService(ProfileClient profileClient,
                              @Value("${rbac.cache.ttl-minutes:5}") int cacheTtlMinutes) {
        this.profileClient = profileClient;
        this.roleCache = Caffeine.newBuilder()
            .expireAfterWrite(cacheTtlMinutes, TimeUnit.MINUTES)
            .maximumSize(10_000)
            .recordStats()
            .build(this::loadRoles);
    }

    private Set<Role> loadRoles(ProfileIdentity identity) {
        return profileClient.fetchRoles(identity);
    }

    public Mono<Boolean> hasAnyRole(ProfileIdentity identity, Set<Role> requiredRoles) {
        if (requiredRoles.isEmpty()) {
            return Mono.just(true);
        }
        return Mono.fromCallable(() -> roleCache.get(identity))
            .subscribeOn(Schedulers.boundedElastic())
            .map(userRoles -> userRoles.stream().anyMatch(requiredRoles::contains));
    }

    public void invalidateCache(ProfileIdentity identity) {
        roleCache.invalidate(identity);
    }

    public CacheStats getCacheStats() {
        return roleCache.stats();
    }
}
```

**File:** `backend-mcp/src/main/java/com/aiadvent/mcp/backend/rbac/service/AuditLogService.java`

```java
@Service
public class AuditLogService {

    private final AuditLogRepository repository;
    private final ExecutorService asyncExecutor = Executors.newFixedThreadPool(4);

    public void logAccess(ProfileIdentity identity, String path, HttpMethod method, String result) {
        asyncLog(() -> {
            AuditLogEntry entry = new AuditLogEntry();
            entry.setActorNamespace(identity.namespace());
            entry.setActorReference(identity.reference());
            entry.setAction("ACCESS");
            entry.setTargetResource(path);
            entry.setHttpMethod(method.name());
            entry.setHttpPath(path);
            entry.setResult(result);
            repository.save(entry);
        });
    }

    public void logDenied(ProfileIdentity identity, String path, HttpMethod method, String reason) {
        asyncLog(() -> {
            AuditLogEntry entry = new AuditLogEntry();
            if (identity != null) {
                entry.setActorNamespace(identity.namespace());
                entry.setActorReference(identity.reference());
            }
            entry.setAction("DENIED");
            entry.setTargetResource(path);
            entry.setHttpMethod(method.name());
            entry.setHttpPath(path);
            entry.setResult("DENIED");
            entry.setMetadata(Map.of("reason", reason));
            repository.save(entry);
        });
    }

    public void logRoleChange(ProfileIdentity actor, ProfileIdentity target,
                              String action, Set<Role> roles) {
        asyncLog(() -> {
            AuditLogEntry entry = new AuditLogEntry();
            entry.setActorNamespace(actor.namespace());
            entry.setActorReference(actor.reference());
            entry.setAction(action); // GRANT or REVOKE
            entry.setTargetProfileNamespace(target.namespace());
            entry.setTargetProfileReference(target.reference());
            entry.setResult("SUCCESS");
            entry.setMetadata(Map.of("roles", roles));
            repository.save(entry);
        });
    }

    private void asyncLog(Runnable task) {
        asyncExecutor.submit(() -> {
            try {
                task.run();
            } catch (Exception e) {
                log.error("Failed to write audit log", e);
            }
        });
    }
}
```

```java
@Component
public class ProfileClient {

    private final WebClient webClient;
    private final String profileServiceUrl;

    public ProfileClient(@Value("${profile.service.url}") String profileServiceUrl,
                         WebClient.Builder webClientBuilder) {
        this.profileServiceUrl = profileServiceUrl;
        this.webClient = webClientBuilder.baseUrl(profileServiceUrl).build();
    }

    public Set<Role> fetchRoles(ProfileIdentity identity) {
        try {
            ProfileResponse response = webClient.get()
                .uri("/api/profiles/{namespace}/{reference}", identity.namespace(), identity.reference())
                .retrieve()
                .bodyToMono(ProfileResponse.class)
                .timeout(Duration.ofSeconds(2))
                .block();

            return response != null && response.roles() != null
                ? response.roles().stream().map(Role::valueOf).collect(Collectors.toSet())
                : Set.of();
        } catch (Exception e) {
            log.error("Failed to fetch roles for {}", identity, e);
            // Fail-closed: return empty set (deny access)
            return Set.of();
        }
    }
}
```

```typescript
// Add to ProfileStoreState
type ProfileStoreState = {
  // ... existing fields
  roleVersion: number; // Incremented on role change
  lastRoleSync: string | null;
  subscribeToRoleChanges: () => () => void;
};

// Add to store implementation
export const useProfileStore = create<ProfileStoreState>()(
  persist(
    (set, get) => ({
      // ... existing initial state
      roleVersion: 0,
      lastRoleSync: null,

      subscribeToRoleChanges: () => {
        const { namespace, reference, channel, devToken } = getActiveProfileSelection();
        if (!namespace || !reference) return () => {};

        const params = new URLSearchParams();
        params.set('namespace', namespace);
        params.set('reference', reference);
        if (channel) params.set('channel', channel);
        if (devToken) params.set('token', devToken);

        const eventSource = new EventSource(
          `${API_BASE_URL}/api/profiles/role-events?${params.toString()}`
        );

        eventSource.addEventListener('role-change', (event) => {
          const data = JSON.parse(event.data);
          set((state) => ({
            profile: state.profile ? {
              ...state.profile,
              roles: data.roles,
              updatedAt: data.timestamp,
            } : null,
            roleVersion: state.roleVersion + 1,
            lastRoleSync: new Date().toISOString(),
          }));
        });

        eventSource.onerror = () => {
          // Auto-reconnect handled by browser, but we log it
          console.warn('Role events connection error, will retry...');
        };

        return () => eventSource.close();
      },
    }),
    // ... existing persist config
  ),
);
```

#### 3.2 Create useRoleGuard Hook
**File:** `frontend/src/lib/useRoleGuard.ts` (new)

```typescript
import { useEffect, useMemo } from 'react';
import { useProfileStore } from './profileStore';

export function useRoleGuard(requiredRole?: string) {
  const roles = useProfileStore((state) => state.profile?.roles ?? []);
  const roleVersion = useProfileStore((state) => state.roleVersion);

  // Subscribe to role changes on mount
  useEffect(() => {
    const unsubscribe = useProfileStore.getState().subscribeToRoleChanges();
    return unsubscribe;
  }, []);

  const hasRole = useMemo(() => {
    if (!requiredRole) return true;
    return roles.some((role) => role === requiredRole);
  }, [roles, requiredRole, roleVersion]);

  const hasAnyRole = useMemo(() => {
    return (roleCodes: string[]) => roles.some((role) => roleCodes.includes(role));
  }, [roles, roleVersion]);

  const hasAllRoles = useMemo(() => {
    return (roleCodes: string[]) => roleCodes.every((role) => roles.includes(role));
  }, [roles, roleVersion]);

  return {
    roles,
    hasRole,
    hasAnyRole,
    hasAllRoles,
    isAdmin: roles.includes('admin'),
    roleVersion,
  };
}
```

```java
@Service
public class RoleEventService {

    private final Map<ProfileIdentity, Sinks.Many<RoleChangeEvent>> subscribers = new ConcurrentHashMap<>();

    public Flux<RoleChangeEvent> subscribe(ProfileIdentity identity) {
        Sinks.Many<RoleChangeEvent> sink = Sinks.many().multicast().onBackpressureBuffer();
        subscribers.put(identity, sink);

        return sink.asFlux()
            .doOnCancel(() -> subscribers.remove(identity))
            .doOnTerminate(() -> subscribers.remove(identity));
    }

    public void notifyRoleChange(ProfileIdentity identity, List<String> roles) {
        Sinks.Many<RoleChangeEvent> sink = subscribers.get(identity);
        if (sink != null) {
            sink.tryEmitNext(new RoleChangeEvent(roles, Instant.now()));
        }
    }

    public void broadcastRoleChange(ProfileIdentity target, List<String> roles) {
        notifyRoleChange(target, roles);
    }

    public record RoleChangeEvent(List<String> roles, Instant timestamp) {}
}
```

```java
@RestController
@RequestMapping("/api/profiles")
public class RoleEventsController {

    private final RoleEventService eventService;

    @GetMapping(value = "/role-events", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public Flux<ServerSentEvent<RoleEventService.RoleChangeEvent>> roleEvents(
            @RequestParam String namespace,
            @RequestParam String reference,
            @RequestParam(required = false) String channel,
            @RequestParam(required = false) String token) {

        ProfileIdentity identity = new ProfileIdentity(namespace, reference);

        return eventService.subscribe(identity)
            .map(event -> ServerSentEvent.builder(event)
                .event("role-change")
                .id(UUID.randomUUID().toString())
                .build())
            .mergeWith(heartbeatFlux());
    }

    private Flux<ServerSentEvent<RoleEventService.RoleChangeEvent>> heartbeatFlux() {
        return Flux.interval(Duration.ofSeconds(30))
            .map(i -> ServerSentEvent.<RoleEventService.RoleChangeEvent>builder()
                .comment("heartbeat")
                .build());
    }
}
```

#### 4.3 Integrate with Admin Role Operations
**File:** `backend-mcp/src/main/java/com/aiadvent/mcp/backend/rbac/service/RoleAssignmentService.java`

```java
@Service
public class RoleAssignmentService {

    private final ProfileClient profileClient;
    private final RoleEventService eventService;
    private final AuditLogService auditService;
    private final ProfileRoleService roleService;

    public Mono<Void> assignRole(ProfileIdentity actor, ProfileIdentity target, Role role) {
        return profileClient.assignRole(target, role)
            .doOnSuccess(v -> {
                // Invalidate cache
                roleService.invalidateCache(target);
                // Notify user
                profileClient.fetchRoles(target)
                    .thenAccept(roles -> eventService.broadcastRoleChange(target, new ArrayList<>(roles)));
                // Audit
                auditService.logRoleChange(actor, target, "GRANT", Set.of(role));
            });
    }

    public Mono<Void> revokeRole(ProfileIdentity actor, ProfileIdentity target, Role role) {
        return profileClient.revokeRole(target, role)
            .doOnSuccess(v -> {
                roleService.invalidateCache(target);
                profileClient.fetchRoles(target)
                    .thenAccept(roles -> eventService.broadcastRoleChange(target, new ArrayList<>(roles)));
                auditService.logRoleChange(actor, target, "REVOKE", Set.of(role));
            });
    }
}
```
